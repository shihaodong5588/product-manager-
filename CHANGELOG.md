## [0.1.0] - 2025-10-11

### 新增
- ✨ 完整的产品管理系统核心功能
- 📋 需求管理模块（创建、编辑、状态跟踪）
- ✅ 任务看板（Kanban 拖拽、任务分配）
- 📊 项目管理（项目创建、成员管理、进度跟踪）
- ⚠️ 风险管理（风险评估、缓解策略）
- 🎯 里程碑管理（进度可视化）
- 👥 团队成员管理（角色权限系统）
- 📈 活动动态（实时活动流）
- 🤖 AI 智能助手（Gemini 2.5 集成）
- 💡 多 AI 模型支持（Flash/Pro/Lite）
- 📝 AI 分析历史记录
- 🎨 完整简体中文界面

### 技术实现
- Next.js 15 + TypeScript
- Prisma ORM + SQLite
- shadcn/ui + Tailwind CSS 4
- Google Gemini API
- @dnd-kit 拖拽功能
